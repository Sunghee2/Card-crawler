# Card-crawler

[네이버 데이터랩](https://datalab.naver.com/local/card_result.naver) 음식점 카드사용통계 크롤링 

- Python, Selenium 사용

- 지역(서울특별시 ~ 제주특별자치도 17개)
- 성별(남자/여자)
- 연령(20대, 30대, 40대, 50대, 60대 이상)
- 카드 사용 기간(2018년 6월 ~ 2019년 6월)
- 음식점 분류(주점, 카페/디저트, 중식, 음식점 기타, 분식, 한식, 어류/해물, 서양식/아시아음식)
